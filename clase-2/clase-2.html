JSX

Es una extensión de Javascript, no de react. La diferencia es que con JSX se tiene autocompletado, manejo de errores. React puede trabajar con Javascript o con JSX.

  • El index.js es una conexión con los elementos, puede seguir siendo .js
  • El documento app.js, se cambia a .jsx

Diferenias entre JS y JSX

  • Cuando se define una clase, ya no es class, sino className
  • Etiqueta input, no tiene etiqueta de cierre, pero en JSX, pasa a ser ""
  • Para insertar código JS en un HTML, se utilizan backticks para insertar texto, ejemplo: innerHTML = `código HTML ${}` ahora, sólo se conservan los curly brackets, así: {user}, cuando se necesita concatenar en el HTML, sí se utilizan las backticks
  • En HTML el atributo style = "nombre" se cambia por {{"nombrePropiedad"}}
  • En general en JSX, cuando las propiedades tienen 2 palabras, serán escritas en camelCase.
Todo el código debe estar dentro de return envuelto entre paréntesis. Ya que React retorna sólo un grupo de elementos.
